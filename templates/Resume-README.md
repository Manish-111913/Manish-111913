## 🛠️ Editing & Compiling LaTeX

If you wish to modify the resume template or build it yourself:

### Option 1: Overleaf (Recommended)
1. Copy the contents of `Resume.tex`.
2. Open [Overleaf](https://www.overleaf.com/).
3. Create a blank project and paste the LaTeX code.
4. Click **Recompile** and download the PDF.

### Option 2: Local LaTeX Installation
You need a LaTeX distribution (e.g., TeX Live or MiKTeX) installed on your machine.
Run the following command:
```bash
pdflatex Resume.tex
```
